Iluminação

Se bem o vídeo tutorial "3D Text in Blender 2.8" demonstra como adicionar o recurso de iluminação na animação realizada pelo blender, a mesma não foi adotada no resultado final deste laboratório por falta de recurso gráfico, pois o computador (notebook) não deu conta do tipo de processamento gráfico.

2D como se fosse 3D

Neste tipo de animações são utilizadas diversas técnicas como perspectiva, sombreamento e iluminação, escala e rotação, camadas, paralaxe, entre outras. Ferramentas de animação, como Adobe After Effects, são também utilizadas graças aos seus recursos específicos para facilitar a aplicação das técnicas que foram citadas. Além disso, linguagens de programação e bibliotecas gráficas podem ser usadas para interativas na web ou em aplicativos. Em resumo, podemos dizer que animações 2D que parecem 3D acabam sendo animações que utilizam todas estas técnicas para criar a ilusão ótica de que o que estamos vendo possui uma terceira camada dimensional.
